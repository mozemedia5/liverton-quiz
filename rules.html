Quiz Rules

Daily Quiz Schedule

  • • Available Time: 9:00 AM to 8:00 PM (EAT)
  • • Quiz Window: 11 hours daily
  • • Days: Monday through Friday only
  • • Questions: Exactly 10 questions per day

Question Format

  • • Time Limit: 15 seconds per question
  • • Options: 4 multiple choice options (A-D)
  • • Retry: One retry allowed per question
  • • Answers: Not revealed until quiz completion

Scoring System

  • • Correct Answer: +3 points
  • • Wrong Answer (first try): 0 points, allowed one retry
  • • Wrong Answer (retry): 0 points
  • • Time Out: 0 points
  • • No Attempt: 0 points
  • • Maximum Daily Score: 30 points

Tie-Breaking Rules

In the event of a tie in total points, the following criteria will be used to determine the winner:

  1. Total Points: Player with higher total points wins
  2. Average Time: Player with lower average response time wins
  3. Tie-Breaker Question: Admin will provide a special question if needed
  4. Consistency: Player with more consistent daily participation

Code of Conduct

  • • Fair Play: No cheating or use of external help
  • • Respect: Treat other participants with respect
  • • Honesty: Do not share answers or collaborate
  • • Integrity: One account per person only
  • • Compliance: Follow all rules and instructions

Grounds for Disqualification

  • • Creating multiple accounts
  • • Using automated tools or bots
  • • Sharing answers with other participants
  • • Attempting to hack or manipulate the system
  • • Violating the code of conduct
  • • Providing false information during registration

Note: Disqualified participants will not be eligible for prizes and may be banned from future competitions.
